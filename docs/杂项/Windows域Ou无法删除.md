# 域Ou无法删除
```
windows server 2008 AD域无法删除组织单位的原因
在 windows server 2008中 Active Directory 域服务中  创建组织单位之后想删除
却发现没有足够的权限、而且是用的 Administrator   
原因：在创建的时候 勾选了 防止对象以意外删除
处理方法：开始 -运行-dsa.msc-右击想要删除的组织单位（OU） -属性- 对象 选卡—取消选项：防止对象意外删除
如果没有对象  选卡 可以 alt+v（菜单里   查看—高级功能）  然后——右击想要删除的组织单位（OU） -属性- 对象 选卡—取消选项：防止对象意外删除
```
